# php-dev-box
A simple vagrant based PHP development environment

## Vagrant cheatsheet
### Destroy vagrant VMs
- vagrant destroy
### Start vagrant VM
- vagrant up
### Providion a started vagrant VM (in other words, run ansible scripts)
- vagrant provision

## SSH connectivity
- IP: 192.168.31.91
